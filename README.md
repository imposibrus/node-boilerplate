# node-boilerplate
## Directory structure:
```
|
|- src/                   - sources folder
|---- bin/
|------- www              - entry point. main executable
|---- controllers/        - controllers for routes
|---- lib/                - common libraries
|---- models/             - DB models
|---- routes/             - site routes
|- node_modules/          - third-party back-end modules
|- public/                - site public files
|- test/                  - unit-tests for some libraries
```
